### **Slide 2: What is Linear Span?**
*(The heading appears. Press Right Arrow for the first bullet)*

**"So, what exactly is a linear span? Formally, it is the set of all possible vectors that can be formed by taking linear combinations of a given set of vectors."**

*(Press Right Arrow for the second bullet)*
**"In many textbooks, this is simply referred to as the 'span' of those vectors."**

*(Press Right Arrow for the formula)*
**"Mathematically, we write it like this: The span of vectors $v_1$ through $v_n$ is equal to $a_1 v_1 + a_2 v_2$, and so on."**

*(Press Right Arrow for the scalar note)*
**"Where the coefficients ($a_1$, $a_2$) can be any scalar numbers—meaning we can stretch, shrink, or flip the vectors however we want."**

*(Press Right Arrow for the visual metaphor)*
**"As you can see in this small diagram, by combining just two base arrows, we can reach entirely new points."**

*(Press Right Arrow for the memorable quote)*
**"To put it in the simplest terms: Linear span just means all the possible results you can make from the vectors you are given."**

*(Press Right Arrow to clear the slide)*

### **Slide 3: Example in $\mathbb{R}^2$**
*(The heading and axes appear)*
**"Let’s look at a concrete example in $\mathbb{R}^2$, which is our standard 2D plane."**

*(Press Right Arrow to show $e_1$ and $e_2$)*
**"We start with the standard basis vectors. $e_1$ pointing exactly 1 unit right, and $e_2$ pointing exactly 1 unit up."**

*(Press Right Arrow to show the formula)*
**"By multiplying them by scalars $a$ and $b$, we can create a formula: $a(1,0) + b(0,1) = (a, b)$."**

*(Press Right Arrow to start the moving dot animation)*
**"Watch what happens if we want to reach a random point, like $(3, 4)$. We simply take 3 steps along $e_1$, and 4 steps along $e_2$. Using just these two building blocks, we can navigate anywhere."**

*(Press Right Arrow for the final conclusion)*
**"Because we can reach literally every coordinate on this graph, we conclude that the Span of $(1,0)$ and $(0,1)$ is the entire $\mathbb{R}^2$ plane."**

*(Press Right Arrow to clear the slide)*

### **Slide 4: Why is Linear Span Important?**
*(The heading appears)*
**"Why do we care about this? Why is linear span important?"**

*(Press Right Arrow for the takeaway)*
**"Because span tells us the total 'reach' of our vectors."**

*(Press Right Arrow for the First Panel - Line)*
**"For example, if you are given just 1 vector, you can only scale it forward or backward. Its span is just a 1-dimensional line."**

*(Press Right Arrow for the Second Panel - Dependent Vectors)*
**"If you are given a second vector, but it points in the exact same direction, it doesn't give you any new reach. These are 'dependent' vectors, and their span is still just that same line."**

*(Press Right Arrow for the Third Panel - Plane)*
**"But, if your second vector points in a completely new direction—meaning they are 'independent'—their combinations break out of the line, and their span becomes an entire 2D plane."**

*(Press Right Arrow to bring in the summary bullets)*
**"So, span tells us the space generated by vectors. It shows us if we are building a line, a plane, or a higher-dimensional space, which is the foundational idea behind basis and dimension."**

*(Press Right Arrow to clear the slide)*
